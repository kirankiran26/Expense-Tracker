# Expense Tracker

## Overview
This project is an Expense Tracker built using React. It allows users to track their expenses by adding new items and amounts, which updates the balance dynamically. The project also utilizes local storage to persist the expense data between sessions.

## Features
- Add new expenses with item names and amounts.
- Dynamically updates the balance based on added expenses.
- Stores expense data locally using browser's local storage.

## Technologies Used
- React
- JavaScript (ES6+)
- HTML5
- CSS (Tailwind CSS for styling)
